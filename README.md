## ⌨기능

- 메모 생성(**C**reate)
- 메모 읽기(**R**ead)
- 메모 수정(**U**pdate)
- 메모 삭제(**D**elete)
- Vuex를 통해 메모 개수 카운트
- Axios로 백엔드 REST api와 연결
